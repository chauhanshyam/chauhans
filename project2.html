Description:

The polber function draws a cartoon polar bear using circles. You give it three inputs: the x and y coordinates to position the bear on the screen, and lor to choose the bear’s color. The function creates the bear’s body as a large colored circle, with a smaller circle above it for the head. It adds ears on the sides, and feet below the body, all in the same color. The eyes are made with grey circles (for the whites) and smaller black circles (for the pupils). A black circle forms the nose, and a slightly larger colored circle below the eyes represents the muzzle. Finally, the head is outlined with a black border to make it stand out. By calling this function with different positions and colors, you can draw multiple colorful polar bears anywhere you like.

The code highlights the use of shapes, colors, borders, and positioning to build a complex graphic.I created a program that displays ways to protect the environment using colors and text. It shows I can make clear and attractive designs with code.
