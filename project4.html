Description:

The onMousePress function handles scene transitions and point scoring based on user clicks. It starts by checking a toggle (app.swi) to move past the home screen, then progresses through different scenes using a counter (app.con). In each scene, the player clicks on objects (ss1, ss2, etc.) that award different point values and advance the game.

As the scenes change, visual states are updated (e.g., HomS(False), sedS(True)), and the final screen is triggered when the last selection is made. This function ties user input to gameplay flow, tracking progress and scoring through a simple click-based interaction system.
