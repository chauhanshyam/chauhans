The Mov function moves a character's body, eye, and tail to follow the mouse. When the mouse moves, the parts shift to match the new x and y position with small offsets. The function also checks if the character enters danger zones—if it does, the player loses a life. If lives run out, it calls a function to show a losing screen.

There’s also a winning zone in the top-right corner. If the character reaches it, the game triggers a victory screen and adds a large number of lives. This shows how the code combines movement, collision checks, and simple game rules to create an interactive experience.
