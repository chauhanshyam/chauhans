Description:

The clos function changes the character’s body color based on how high it moves on the screen, using the y-position to indicate proximity. As the character gets closer to the top, the color shifts from yellow to red. If lives are already lost, it also updates a message with phrases like "very far" or "almost there" to reflect how close the player is.

The onMouseMove function lets the character follow the mouse, adjusting the body, eye, and tail positions with slight offsets. It also checks if the character enters any danger zones—if so, a life is lost. When lives run out, a losing screen is triggered. This ties movement, danger detection, and visual feedback into the gameplay.
